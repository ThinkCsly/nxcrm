## 关于 Nxcrm
 ---

Nxcrm 是宁夏南相开发的客户管理系统，包含了线索，商机，合同，收款，客户，附件，联系人，跟进动态等功能。囊括可客户管理所需的大多数功能。后期版本我们将增加小程序功能。敬请期待。  
  
    
##### DEMO：
https://crm.demo.nx.tt/admin
